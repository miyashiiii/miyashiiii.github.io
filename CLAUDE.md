# Claude Code Actions 設定

## 言語設定

- すべてのコメント、レスポンス、メッセージは**日本語**で行ってください。
- コードコメントも日本語で

## コミットメッセージ

- コミットメッセージは日本語で、変更内容を簡潔に説明してください
- prefixは不要
- フォーマット例: `○○機能を追加` `○○のバグを修正` `○○のドキュメントを更新`

## UI変更時のビジュアル確認ワークフロー

UI（ページ、コンポーネント）を変更した場合は、必ず **screenshot スキル** を使ってビジュアル確認を実施する。

### Skillを使った自動確認（推奨）

- UI変更を検知したら、自動的に `screenshot` スキルが起動
- デスクトップ（1920x1080）、タブレット（768x1024）、モバイル（375x667）の3サイズでスクリーンショットを取得
- Playwright MCPを使って各デバイスサイズをエミュレート
- レスポンシブ対応、レイアウト、画像読み込み、余白などを自動レビュー

### 手動実行

必要に応じて以下のコマンドでも実行可能：
- `/screenshot` - Skillを手動起動
- `npm run screenshot:tmp` - Node.jsスクリプトで実行（/tmpに保存）

### 前提条件

- 開発サーバーが http://localhost:3001 で起動していること（`npm run dev`）
- Playwright MCPサーバーが有効化されていること（`.claude/mcp.json`）
